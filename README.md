# Signals package
This is a simple package born from regularly reusing similar functions to generate:
- smoothed data
- derivative with smoothing the data
- finding peaks in curves
- background estimation and correction for various data sets
- an extension of apply to use pattern searching in applying a function and assign to
  store the results.

These problems are rather generic for one dimensional data sets, and after feeling
redundant to always rewrite the same functions, there were collected here into a small
package.
